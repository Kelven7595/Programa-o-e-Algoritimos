## 🔍 Visão Geral

A programação está presente em praticamente todas as áreas da tecnologia. Este repositório foi criado para registrar minha evolução durante a disciplina, armazenando exercícios, desafios e projetos que auxiliam no desenvolvimento do pensamento lógico e computacional.
